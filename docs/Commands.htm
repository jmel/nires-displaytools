NIRES Command List
     
Field Viewer Commands
Cursors
Action Command Description
Center a cursor on object                cu cent n  For cursor number n (integer), finds the centroid of the light under the cursor and moves the cursor to that position
Delete a cursor               cu del n  Delete cursor number n (integer)
Delete all cursors               cu del all Delete all cursors
Display a cursor               cu disp n Display cursor number n (integer)
Display all cursors               cu disp all Display all cursors
Move a cursor to coordinate                   cu mv n x y Move cursor number n (integer) to pixel coordinates x (float) and y (float)
Move a cursor to slit position                   cu mv n slit m Move cursor number n (integer) to slit position m (integer)
Display Images
Action Command Description
Display Linear Stretch lindispv n m
Apply a linear stretch from n (integer) to m (integer)
Display picture dpv n Display a field viewer image number n (integer)
Display picture current dpcv  Display last picture taken 
Subtract pictures and display pdiffv n m Subtract image # m (integer) from image # n (integer) and display
Move Telescope
Action Command Description
move east north en e n Move the telescope e (float, arcsec) in RA and n (float, arcsec) in Dec
move from cursor to cursor mt cu n m Moves the telescope so that an object located in cursor box n (integer) will now land in cursor box m (integer)
Obtain Images
Action Command Description
Set the exposure time tintv t Sets the exposure time to t (float, sec)
Set the Fowler sampling fowlerv n Sets the number n (integer) of Fowler Samples
Set the frame number framev n Sets the frame number n (integer) for the next  exposure
Set the number of coadds coaddv n Sets the number of coadds, n (integer)
Set the object name objectv s Sets the name of the object in the viewer to 's' (string)
Set the object name for both the viewer and spectrograph object s Sets the name of the object in the viewer and the spectrograph to 's' (string)
Set the sampling mode sampmodev n Sets the sampling mode n (integer):
1=single sampling,
2=correlated double sampling,
3=multiple correlated double sampling
Show the current image settings showv Shows all of the settings for the current exposure
Show the current image setting in an auto-updated window xshowv Pops up window that shows the current image settings and auto-updates the display as the settings change
Take an Image goiv Takes the next exposure

Markers
Action Command Description
Show observed wavelength dw Overlay markers that show the observed wavelength
Show the rest wavelength dz z Displays markers that show the rest wavelength given an input redshift, z (float)
Show typical emission-line features dez z Displays markers that show the typical nebular emission features at a given redshift, z (float)
